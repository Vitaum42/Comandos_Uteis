## ⚠️ Boas Práticas Gerais

- Sempre leia o que um comando faz antes de executá-lo com `sudo`
- Use `man <comando>` para acessar o manual de qualquer ferramenta
- Faça backups antes de modificar arquivos do sistema
- Mantenha o sistema atualizado com `sudo apt update && sudo apt upgrade`
- Em caso de dúvida, prefira fazer menos com mais segurança
